\section{Calculate the Potential Field}

Das Potential auf einer der ebenen Fläche wird wie folgt berechnet:
\begin{lstlisting}
% Koordinaten in der X,Y-Ebene
[X,Y] =  meshgrid(-dim(1)/2:0.1:dim(1)/2,...
				  -dim(2)/2:0.1:dim(2)/2);
% Radialer Abstand
r = sqrt(X.^2 + Y.^2 + z^2);
% Potential
Phi = (rho_medium * I) ./ (4 * pi .* r);
\end{lstlisting}
